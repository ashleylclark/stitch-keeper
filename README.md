# Stitch Keeper

A personal crochet/knitting stash, pattern, and project tracker

## Development

```bash
npm install
npm run dev
```

- Vite frontend: `http://localhost:5173`
- Node API: `http://localhost:3001`

## Persistence

- SQLite database file: `data/hook-stash.db`
- API entrypoint: `server/index.js`
- The schema and seed data are created automatically on first startup

App UI and API: `http://localhost:8080`
